# DESIGN.md — Ник Меньшов · Landing Style Guide

> Single source of truth. Direction: **modern fashion / editorial luxury** (не «сдержанный минимализм» — слоистый, журнальный, с японскими нюансами).
> Status: CP1 LOCKED (hero), pending go for full build.

## 1. Brand essence
- **Имя:** Ник Меньшов. **Роль: Sound Director — DJ.** Международный уровень, **с 2017**.
- **Тон:** уровень считывается визуалом/кейсами/портфолио, не копирайтом. Без кричащих фраз и AI-клише. **Участие AI не афишируется** — убираем AI-якоря в текстах и визуале.
- **Языки:** English C2 · 日本語 базовый.
- **Регалии:** Член жюри **Top100 Awards** (нужна точная ссылка) · Участник **Top15 Moscow** (ссылка есть).
- Японский нюанс: глиф 音 (звук) как брендовый знак (печать/водяной знак), не как «декор ради декора».

## 2. Palette — A · Vermilion Noir (LOCKED)
```css
--fg:    oklch(96% 0.010 85);   /* ivory */
--muted: oklch(80% 0.012 80);
--faint: oklch(66% 0.012 80);
--accent:oklch(67% 0.17 32);    /* vermilion */
--scrim: oklch(15% 0.012 50);   /* warm graphite-black overlay поверх видео */
--hair:  oklch(96% 0.01 85 / 0.18);
```
- Свет: ivory-текст поверх затемнённого тёплого видео-скрима. Акцент (вермильон) — точечно: CTA, маркеры, печать, ссылки.
- (Варианты B · Champagne и C · Ink & Sakura реализованы как переключатель в концепте — не используются по умолчанию.)

## 3. Typography
- **Display:** `Prata` (Didone, кириллица) — имя, заголовки. Fashion-editorial.
- **UI / body / labels:** `Archivo` (кириллица, variable) + `Archivo Expanded` для акцентных лейблов. Лейблы — UPPERCASE, трекинг 0.3em.
- **音** — `Noto Serif JP` (только глиф, subset).
- Имя: `clamp(2.8rem, 0.4rem + 10vw, 9.5rem)`, line-height 0.86. **Без цветного выделения частей слова** (имя одним цветом).
- ⚠️ В CSS-`clamp()` ВСЕГДА пробелы вокруг `+`/`-` (иначе правило невалидно → наследуется огромный размер).

## 4. Hero composition (готово в концепте)
- **Фон: полноэкранный шоурил** — кроссфейд-цикл reel1→reel4 (две `<video>`-слоя), poster для мгновенной отрисовки. Поверх — scrim-градиент + grain.
- Слои/сложность: вращающаяся **печать «ЧЛЕН ЖЮРИ · TOP100 AWARDS»** (центр 音), бегущий **тикер** клиентов, вертикальный текст справа, hairline-сетка, мета-блок (Доступен / Moscow—International / Est. 2017), имя Prata, tagline + мета (Москва·с2017 · языки · Участник Top15 ↗), CTA.

## 5. Showreel (assets)
- Исходники: Я.Диск (8 видео, портрет 720×1280, iPhone). Отобраны сценические клипы: **2909** (DJ за пультом), **2910** (DJ+толпа+дым), **2911** (сцена+зал), **2907** (воздушная гимнастка, премиум-ивент). Отброшены: 2906 (лаунж), 2908 (тёмная дверь).
- Транскод: `avconvert` (VideoToolbox) → H.264 (avc1), ~4с каждый, web-safe для всех браузеров. Файлы `concept/media/reel1..4.mp4` (~4.8MB суммарно) + `poster.jpg`.
- Инструменты на машине: `avconvert` (транскод/тримминг), `qlmanage` (постеры/превью). Системного `ffmpeg` нет; Playwright-ffmpeg не читает HEVC.

## 6. Motion — медленно, дорого
- Кроссфейд шоурила ~0.9с; reveal-stagger; вращение печати; тикер; hover-подчёркивания; lift CTA.
- transform/opacity/clip-path · 60fps · без layout shift · полный `prefers-reduced-motion`.

## 7. Sections (one-page, RU, sticky-nav)
1. **Hero** (готово).
2. **Обо мне** — спокойное био, языки, с 2017, международный, регалии.
3. **Саунд-директор** *(новый раздел)* — вводный текст (см. CONTENT.md) + сетка **кейсов** (задача → звук → результат). Нужны материалы (CP2).
4. **Сеты / Портфолио** — видео-рилы + **стилизованный аудиоплеер** (миксы выдаст клиент позже), lazy.
5. **Райдер** — пункты + «Скачать райдер (PDF)».
6. **Даты и бронирование** — календарь + форма (serverless).
7. **Контакты** — Telegram, канал, Instagram, tel:+7-910-572-25-08.
- CTA «Забронировать дату» — hero/nav/footer (≤2 клика).

## 8. Media gaps (CP2 — от клиента)
- Фото (портрет «Обо мне», кадры для кейсов/галереи) — на Я.Диске фото нет.
- Миксы для аудиоплеера.
- Материалы кейсов саунд-дирекции.
- Точная ссылка/профиль **Top100 Awards**.

## 9. A11y & perf
- Контраст AA+ (ivory на тёмном скриме). Клавиатура, фокус, reduced-motion.
- Lighthouse ≥ 90; адаптив 320–1920; lazy media; видео — poster + preload-метаданные, пауза вне вьюпорта; ≤2 шрифт-семьи (+Noto JP subset).
- Мобайл: бургер-меню (в концепте — to-do на вёрстке).
